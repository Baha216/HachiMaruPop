<h2>「はちまるポップ」について</h2>
<p>こちらはGoogleFonts様に提供するフォントです。<br>
1980年代頃の日本で流行した丸文字からアイディアを得て、現代風なデザインに落とし込みました。<br>
ライセンスはSIL Open Font License Version 1.1に準拠します。</p>
